Game Building

My second experience in the video game sector, at Quantic Dream. This was in the context of their fourth game, a AAA production on PS3: Beyond: Two Souls.

The game building consists in putting together the different elements of a game (objects, characters, animations, dialogues) and to script the player's interactions with all these elements, as well as the AI of the non-playable characters.

Mission of the Game Builder:

  • Put together the game elements, consistently with the game design.
  • Script up all available interactions in each game level.
  • Create the animated sequences, integrate sound and dialogs.
  • Script NPCs' artificial intelligence.
  • Ensure all elements remain consistent in each level.
